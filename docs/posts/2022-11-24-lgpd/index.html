Lei Geral de Proteção de Dados - Lei 13.709/2018 vigente a partir de setembro de 2020.

GDPR - A União Européia só fará negócio com países com legislação de mesmo nível.

GDPR - General Data Protection Regulation - Regulamento Geral de Proteção de Dados 2016 entrou em vigor em 2018

Lei de proteção de dados pessoais e não de proibição de dados pessoais.

Se conecta com o código de defesa do consumidor, por isso deve-se olhar as leis verticais que se aplicam. Por exemplo, CFM (Conselho Federal de Medicina) para questões de Tele medicina, ou BACEN para questões financeiras.

Diferença do regulamento europeu: proteção do crédito.

Exceção de consentimento: “ter certeza de que aquela pessoa é aquela pessoa”.
