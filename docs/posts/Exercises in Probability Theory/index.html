1. Amostragem

Como gerar números aleatórios a partir de um conjunto? Para jogar na mega-sena é preciso escolher 6 números dentre 60 de 00 até 59. Como selecionar em R 6 números aleatórios para jogar na mega-sena?

set.seed(123)
sample(0:59, 6)
[1] 30 14 50 13  2 41

Como escolher 2 frutas aleatórias do conjunto: pêra, maçã, banana, mamão e jabuticaba?

set.seed(123)
frutas = c("pêra", "maçã", "banana", "mamão", "jabuticaba")
sample(frutas, 2)
[1] "banana" "maçã"

2. Variância

A variância de uma variável discreta Y é descrita como:

\[ \sigma_Y^2 = Var(Y) = E[(Y - \mu_y)^2] = \sum_{i = 1}^{k}(y_i - \mu_i)^2p_i \] sendo o desvio padrão de Y, denominado \(\sigma_Y\), a raiz quadrada da variância.

Importante notar que R não possui uma função que calcula a variância. A função var() calcula a variância de uma amostra cuja equação é dada por:

\[ \sigma_Y^2 = {1 \over (n - 1)} \sum_{i = 1}^{n}(y_i - \overline{y})^2 \] que é diferente da equação da variância de uma população dada por:

\[ Var(Y) {1 \over N} \sum_{i = 1}^{n}(y_i - \mu_Y)^2 \] A diferença fica clara quando comparamos os resultados para um exemplo de um jogo de dados:

\[ Var(Y) = {1 \over 6} \sum_{i = 1}^{6}(d_i - 3.5)^2 = \]

\[ = {1 \over 6}[(1 - 3.5)^2 + (2 - 3.5)^2 + (3 - 3.5)^2 + (4 - 3.5)^2 + (5 - 3.5)^2 + (6 - 3.5)^2] = \]

\[ = {1 \over 6}[6.25 + 2.25 + 0.25 + 0.25 + 2.25 + 6.25] = 2.92 \]

enquanto que em R a variância da população:

# variância:
y <- 1:6
y_mean <- mean(y)
var_y <- sum((y - mean(y))^2)/6
var_y
[1] 2.916667

e a variância da amostra:

# variância da amostra:
var(1:6)
[1] 3.5

3. Função de Densidade Probabilística (FDP)

Considere a variável aleatória X com função de densidade probabilística igual a:

\[ f(x) = \frac{x}{4} e^{-x^2/8}, x > 0 \]

Defina uma função em R com esta equação:

pdf_x <- function(x) {
  (x/4) * (exp(-1*(x^2)/8))
}

pdf_x(0)
[1] 0
pdf_x(1)
[1] 0.2206242
pdf_x(c(2,4,6))
[1] 0.30326533 0.13533528 0.01666349
pdf_x(Inf)
[1] NaN

Para que f(x) seja uma FDP sua integral sobre o domínio deverá ser 1:

\[ \int_{0}^{\infty} f(x)dx = 1 \] Em R, a função integrate permite calcular a integral passando como parâmetros a função e os limites:

integrate(pdf_x, 0, Inf)$value
[1] 1

O parâmetro value após a função permite capturar o retorno da mesma. Pesquise por integrate no help do R.

4. Valor Esperado e Variância

Vamos calcular o valor esperado e a variância de uma variável contínua considerando a função f(x) definida anterioremente.

O valor esperado é dado por (Lembrando que a função está definida para x >= 0):

\[ E(x) = \int_{0}^{\infty}x f_X(x)dx \]

Utilizaremos a função pdf_x definida no exemplo anterior.

ex <- function(x){
  x*pdf_x(x)
} 
expected_value <- integrate(ex, 0, Inf)$value

A variância é obtida por:

\[ Var(X) = E(X^2) - E(X)^2, onde E(X^2) = \int_{0}^{\infty}x^2f_X(x)dx \]

ex2 <- function(x){
  x^2*pdf_x(x)
}
variance <- integrate(ex2, 0, Inf)$value - expected_value^2
variance
[1] 1.716815
  1. Distribuição Normal

Em R a distribuição normal padrão possui uma série de funções:

  • rnorm(n, mean = 0, sd = 1)

Retorna n elementos de uma distribuição normal com \(\mu\) = 0 e \(\sigma\) = 1:

rnorm(3)
[1] 1.55870831 0.07050839 0.12928774
rnorm(3, mean = 1, sd = 1.5)
[1]  3.5725975  1.6913743 -0.8975919
  • dnorm(x, mean = 0, sd = 1, log = FALSE)

Retorna a densidade da distribuição para os valores de x:

dnorm(3)
[1] 0.004431848
dnorm(1:3)
[1] 0.241970725 0.053990967 0.004431848
  • pnorm(q, mean = 0, sd = 1, lower.tail = TRUE, log.p = FALSE)
  • qnorm(p, mean = 0, sd = 1, lower.tail = TRUE, log.p = FALSE)

pnorm calcula a probabilidade: \(P(x \leq q)\) para um valor de q.

\(P(x \leq -1)\)

pnorm(-1)
[1] 0.1586553

qnorm calcula o valor de q para a probabilidade indicada: $P(x q) = 0.975 $P(x q) = 0.975

qnorm(0.975)
[1] 1.959964
